# Install Zircon

## Installing Zircon - Command Line
Dependencies:
- Git
- NodeJS

### 1. Get Zircon:
```sh
git clone --single-branch --branch main https://github.com/SolarFlurry/zircon
cd zircon
```

### 2. Build:
```sh
npm run build
npm link
```

### Updating Zircon:
```sh
npm run update
```
